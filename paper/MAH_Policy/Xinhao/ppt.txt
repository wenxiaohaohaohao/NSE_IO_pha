%------------------------------------------------
\begin{frame}{1. Main Question}
\begin{itemize}
    \item This paper asks a narrow but two-layer question:
    \[
    \text{Does MAH raise original-drug innovation by changing commercialization routes?}
    \]
    \item Mechanism object: \textbf{commercialization assignment}.
    \item Final result object: \textbf{realized original-drug innovation}.
    \item We do not model MAH as a generic pro-innovation policy or a direct scientific productivity shock.
\end{itemize}

\vspace{0.25cm}
\begin{block}{Object hierarchy}
    \[
    \text{Mechanism: assignment} \;\Rightarrow\; \text{Result: original-drug realization}
    \]
\end{block}
\end{frame}

%------------------------------------------------
\begin{frame}{2. Motivation: Why Commercialization Is the Bottleneck}
\begin{itemize}
    \item In pharmaceuticals, invention and commercialization need not happen inside the same firm.
    \item A scientifically promising idea may still fail if the innovator lacks downstream manufacturing capability.
    \item Before MAH, a research-originating firm often faced a difficult choice:
    \begin{enumerate}
        \item integrate downstream,
        \item transfer effective control, or
        \item fail to commercialize.
    \end{enumerate}
    \item This means the bottleneck is often \textbf{organizational implementation}, not only scientific discovery.
\end{itemize}

\vspace{0.3cm}
\textbf{Core intuition:} MAH matters if it changes which route from idea to market is feasible.
\end{frame}

%------------------------------------------------
\begin{frame}{3. What MAH Changes Institutionally}
\begin{itemize}
    \item MAH decouples, in legal form, \textbf{authorization holding} from \textbf{in-house production}.
    \item A holder may retain authorization while commissioning production to a qualified outside manufacturer.
    \item But this is not simple outsourcing:
    \begin{itemize}
        \item the relationship is formalized,
        \item the holder still bears meaningful responsibility,
        \item quality and compliance obligations remain central.
    \end{itemize}
    \item So MAH does not remove regulation; it creates a more workable \textbf{external commercialization route}.
\end{itemize}

\vspace{0.3cm}
\textbf{Takeaway:} MAH changes the organizational feasibility of commercialization.
\end{frame}

%------------------------------------------------
\begin{frame}{4. Policy Primitive and Identification Boundary}
\begin{block}{Policy primitive}
We model MAH as lowering the route-specific friction of external commercialization:
\[
\tau_{ext}(1) < \tau_{ext}(0).
\]
\end{block}

\begin{itemize}
    \item \(\tau_{ext}\) is not demand, not scientific productivity, and not a universal policy wedge.
    \item It is the governance and compliance friction of external commercialization with holder responsibility.
    \item Empirically, we can map outcomes to this mechanism, but we do not claim to directly observe \(\tau_{ext}\).
\end{itemize}

\vspace{0.3cm}
If MAH lowers \(\tau_{ext}\), then external commercialization becomes more attractive.
\end{frame}

%------------------------------------------------
\begin{frame}{5. Model Architecture}
\begin{itemize}
    \item Time is discrete.
    \item Firms are heterogeneous with state \(z\), and choose across two project directions:
    \[
    k\in\{G,O\}\quad (\text{generic},\;\text{original-drug}).
    \]
    \item Three organizational forms:
    \begin{enumerate}
        \item \textbf{Type A:} integrated firms
        \item \textbf{Type B:} research-specialized firms
        \item \textbf{Type C:} manufacturing-specialized firms
    \end{enumerate}
    \item The model includes:
    \begin{itemize}
        \item endogenous entry and exit,
        \item organizational switching,
        \item a contract-manufacturing market,
        \item commercialization assignment for original-drug projects.
    \end{itemize}
\end{itemize}

\vspace{0.3cm}
\textbf{Important design choice:} MAH acts on implementation value, then feeds back into innovation direction.
\end{frame}

%------------------------------------------------
\begin{frame}{6. Economic Roles of the Three Types}
\begin{columns}[T]
\column{0.32\textwidth}
\begin{block}{Type A}
Integrated firms
\begin{itemize}
    \item generate ideas
    \item implement internally
    \item benchmark internal route
\end{itemize}
\end{block}

\column{0.32\textwidth}
\begin{block}{Type B}
Research-specialized firms
\begin{itemize}
    \item generate ideas
    \item do not have full internal manufacturing
    \item central to the mechanism
\end{itemize}
\end{block}

\column{0.32\textwidth}
\begin{block}{Type C}
Manufacturing-specialized firms
\begin{itemize}
    \item provide qualified external production
    \item serve contract-manufacturing market
\end{itemize}
\end{block}
\end{columns}

\vspace{0.3cm}
\textbf{Interpretation:} Type B firms are where MAH should matter most.
\end{frame}

%------------------------------------------------
\begin{frame}{7. Timing and the Core State Variable}
\begin{itemize}
    \item Each incumbent starts the period with organizational form \(s\in\{A,B,C\}\) and capability \(z\).
    \item Static production choices are optimized out; dynamic decisions are central.
    \item Within-period dynamic sequence:
    \begin{enumerate}
        \item choose project direction and effort,
        \item assign original-drug projects to commercialization routes,
        \item choose stay / switch / exit.
    \end{enumerate}
    \item Baseline uses one-dimensional state:
    \[
    z\in Z\subset\mathbb{R}_+.
    \]
\end{itemize}

\vspace{0.25cm}
\textbf{Why this baseline?} Tractable enough for estimation now, expandable to richer states later.
\end{frame}

%------------------------------------------------
\begin{frame}{8. Type-B Direction Choice Block}
\small
Type-B firms allocate effort across two directions:
\[
\Pi_B(z;M,p_m)
=
-f_B+
\max_{x_B^G,x_B^O\ge 0}
\left\{
x_B^G\Psi_B^G(z)
+
x_B^O\Psi_B^O(z;p_m,M)
-
C_B(x_B^G,x_B^O)
\right\}.
\]

\begin{itemize}
    \item \(x_B^G\): generic-oriented effort.
    \item \(x_B^O\): original-drug-oriented effort.
    \item \(C_B(\cdot)\): joint convex cost with resource rivalry across directions.
\end{itemize}

\vspace{0.2cm}
\textbf{Discipline condition:}
\begin{itemize}
    \item MAH does not directly shift \(\Psi_B^G\).
    \item MAH shifts \(\Psi_B^O\) via external commercialization value.
\end{itemize}
\end{frame}

%------------------------------------------------
\begin{frame}{9. Original-Drug Commercialization Assignment}
\small
For original-drug projects, Type-B firms choose a route:
\[
H_O=\{ext,\; int,\; tr,\; ab\}.
\]

External route value:
\[
H_{B,O}^{ext}(z,p_m;M)=\lambda_{ext}(z,p_m)R_B^O(z)-p_m-\tau_{ext}(M).
\]

Internal route value:
\[
H_{B,O}^{int}(z)=\lambda_{int}(z)R_B^O(z)-\kappa_I(z).
\]

Per-project original-drug value:
\[
\Psi_B^O(z;p_m,M)=
\max\left\{
H_{B,O}^{ext},\,
H_{B,O}^{int},\,
H_{B,O}^{tr},\,
0
\right\}.
\]

\begin{itemize}
    \item MAH enters directly only through \(H_{B,O}^{ext}\).
    \item This preserves the institutional mechanism and avoids a generic policy shock interpretation.
\end{itemize}
\end{frame}

%------------------------------------------------
\begin{frame}{10. Why MAH Raises Type B Research}
\small
Define the Type-B original-drug effort share:
\[
\omega_B^O(z;M,p_m)=
\frac{x_B^{O*}}{x_B^{G*}+x_B^{O*}}.
\]

\begin{block}{Main logic}
\[
\tau_{ext}\downarrow
\;\Rightarrow\;
H_{B,O}^{ext}\uparrow
\;\Rightarrow\;
\Psi_B^O\uparrow
\;\Rightarrow\;
x_B^{O*}\uparrow,\;\omega_B^O\uparrow
\]
\end{block}

\begin{itemize}
    \item Commercialization value rises first, then research incentives adjust.
    \item The adjustment is not only level \((x_B^{O*})\), but also direction \((\omega_B^O)\).
    \item So MAH raises Type-B research through commercializability, not direct science productivity.
\end{itemize}
\end{frame}

%------------------------------------------------
\begin{frame}{11. Comparative Statics: What Moves with MAH?}
\small
\begin{block}{Direct effect at fixed \((z,p_m)\)}
\[
\tau_{ext}\downarrow
\Rightarrow
H_{B,O}^{ext}\uparrow
\Rightarrow
\Psi_B^O\uparrow.
\]
\end{block}

\begin{itemize}
    \item \textbf{Route margin:} the external-route choice set expands.
    \item \textbf{Direction margin:} \(x_B^{O*}\uparrow\) and \(\omega_B^O\uparrow\), potentially crowding out generic effort.
    \item \textbf{Entry margin:} \(V_B\uparrow\), so research-oriented entry becomes more attractive.
    \item \textbf{GE caveat:} producer-side Type-C profits are sign-ambiguous because \(p_m\), entry, and costs adjust endogenously.
\end{itemize}

\vspace{0.1cm}
\textbf{Reading:} MAH shifts one route-specific friction; broad industry outcomes follow through equilibrium propagation.
\end{frame}

%------------------------------------------------
\begin{frame}{12. Dynamic Structure and Equilibrium}
\small
Firms can stay, switch organizational form, or exit.

Bellman equation:
\[
V_s(z;M)=
\max\left\{
0,\;
\max_{j\in\mathcal{J}(s)}
W_{sj}(z;M)
\right\},
\qquad s\in\{A,B,C\}.
\]

where
\[
W_{sj}(z;M)=
\Pi_j(z;M,w,p_m)
-\kappa_{sj}(z;M)
+\beta E[V_j(z';M)\mid z,j].
\]

Contract-manufacturing market clearing:
\[
S_m(p_m,w;M)=D_m(p_m;M).
\]

\begin{itemize}
    \item Entry, exit, switching, and prices are all endogenous.
    \item We track both mechanism objects (route assignment, direction choice) and final innovation outcomes.
\end{itemize}
\end{frame}

%------------------------------------------------
\begin{frame}{13. Main Predictions}
\begin{block}{First-layer (main outcomes)}
\begin{itemize}
    \item external route use for original-drug projects rises,
    \item original-drug-oriented effort \(x_B^{O*}\) and share \(\omega_B^O\) rise,
    \item realized original-drug outcomes rise: \(N_O\), \(S_O\), and conversion \(Conv_O\).
\end{itemize}
\end{block}

\begin{block}{Second-layer (supporting margins)}
\begin{itemize}
    \item research-oriented entry margin becomes more attractive,
    \item organizational transitions \((A,B,C)\) may adjust,
    \item producer-side Type-C effects are not guaranteed monotone in general equilibrium.
\end{itemize}
\end{block}
\vspace{0.1cm}
\textbf{Interpretation:} We prioritize innovation outcomes first, then use reallocation margins as mechanism validation.
\end{frame}

%------------------------------------------------
\begin{frame}{14. Empirical Mapping and How We Proceed}
\begin{itemize}
    \item \textbf{Module A (current approval-side data):}
    \begin{itemize}
        \item holder-producer split as route-use proxy,
        \item realized original-drug counts and shares,
        \item baseline reduced-form and baseline moments.
    \end{itemize}
    \item \textbf{Module B/C (future upgrades):}
    \begin{itemize}
        \item application-to-approval conversion and explicit entrusted production,
        \item capability-state construction, firm entry, and richer type mapping.
    \end{itemize}
    \item Empirical claims remain modular: stronger data tighten the same model objects.
\end{itemize}

\vspace{0.2cm}
\textbf{How we proceed (for this project stage):}
\begin{itemize}
    \item finish Module-A baseline facts and reduced forms,
    \item solve stationary equilibrium and estimate baseline moments,
    \item add Module-B/C data for conversion, capability, and entry,
    \item run counterfactuals after full moment expansion.
\end{itemize}
\end{frame}

%------------------------------------------------
\begin{frame}{15. Data-Estimation GAP: What We Have vs. What We Need}
\small
\begin{block}{What current Module-A data identifies well}
\begin{itemize}
    \item holder-producer split (route-use proxy),
    \item realized original-drug counts and shares,
    \item baseline pre/post moments for reduced-form discipline.
\end{itemize}
\end{block}

\begin{block}{Main GAP for structural parameter estimation}
\begin{itemize}
    \item \textbf{Conversion denominator missing:} approval-only data cannot pin down application-to-approval conversion moments.
    \item \textbf{Capability state missing:} limited pre-reform firm capability measures for identifying state transition and type mapping.
    \item \textbf{Route observability gap:} split proxy exists, but explicit entrusted-production indicators are still limited.
\end{itemize}
\end{block}

\textbf{Near-term fix:} keep baseline estimation modular now, then expand the moment set when Module-B/C merges are complete.
\end{frame}

%------------------------------------------------
\begin{frame}{16. What We Need Feedback On}
\begin{itemize}
    \item Is the object hierarchy clear enough: assignment as mechanism, original-drug innovation as final object?
    \item Is the direction-choice block \((x_B^G,x_B^O)\) the right minimal extension at current complexity?
    \item Is the Module A to Module B/C empirical roadmap disciplined and convincing for identification?
\end{itemize}

\vspace{0.4cm}
\begin{block}{Bottom line}
MAH matters because it changes \emph{how} ideas become products and \emph{which} ideas firms choose to pursue.
\end{block}
\end{frame}
